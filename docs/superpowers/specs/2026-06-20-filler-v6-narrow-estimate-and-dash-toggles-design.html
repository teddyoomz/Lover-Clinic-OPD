Purpose / Problem

The estimate range read too wide to be credible — e.g. 7"/30cc → girth 12.7–13.6 cm → condom "XL 60 – 68" (~4 sizes). User: "เหวี่ยงกว้าง … ดูไม่น่าเชื่อถือ … ตรวจสอบสูตรทั้งวิทยาศาสตร์/คณิตศาสตร์ + มาตรฐานถุงยาง + บทวิชาการรองรับ." Plus 2D-illustration UX: toggle the dashed outlines, relocate the legend, stronger red glow, smart auto-scale across devices.

Formula audit (cited)

  • Geometry CORRECT. ISO 4074: nominal width = ½ circumference → our width_mm = girth_cm × 5 is exact. Cross-section model C₁ = √(C₀² + 4πV/L) is sound (A = C²/4π, ΔA = V/L).
  • k-multipliers were too high + too wide. RCT (16.4 mL HA, baseline girth 7.87 cm): +2.5 cm @1mo, +1.9 @6mo, +1.4 @18mo → implied efficiency k ≈ 1.7–2.5 (peak→durable). Old k = 2.37–3.32 overshoots (3.32 > even the 1-mo peak); the 40% spread caused the wide range.
  • A1 fix: recalibrate to k = 1.8–2.3 (durable→peak). Same 7"/30cc → girth 12.1–12.6 cm → XL 60. Narrow + clinically anchored.
  • Honest caveat: real per-individual SD is ±1.3 cm (large) — the narrow band is the typical/expected result; the existing "ผลลัพธ์แตกต่างกันในแต่ละบุคคล" disclaimer covers individual variation.
  • Condom ladder: real ISO products span 45–72 mm (MyONE to 64; precise sizing to ~72). So 66–72 are REAL sizes → "เกินมาตรฐาน 🔥" now fires only past 72.

Citations: HA penile augmentation RCT (J Clin Med 2020, PMC7230452) · ISO 4074 · nominal width = ½ circumference · 45–72 mm real range

Flow

#Estimate (A1) + ladder
1estimate(): girth gain g = √(C₀²+4πV/L) − C₀; deltaLow = K_REALISTIC(1.8)·g, deltaHigh = K_OPTIMISTIC(2.3)·g.
2condomForGirth(girth): req = girth×5. ≤64 → named ladder (floor). 66–72 → real size, number label, beyond:false (+N ขนาด). >72 → beyond:true (เกินมาตรฐาน 🔥).
3ResultCard: narrower girth range + condom size; delta = เกินมาตรฐาน only when >72.
#2D dash toggle (B)
1FillerGraphic2D holds showAfter + showBaseline state (default true).
2Bottom row: color-keys left · 2 toggle chips right (dashed swatch + label = the legend). Tap → flip state.
3Red dashed outline (×2 svgs) + faint baseline (×2) render conditionally on the state. Red outline keeps breathe + a STRONGER glow. 44px tap targets, flex-wrap auto-scale.

Decisions (locked via Q&A)

  • A1: recalibrate k 2.37/3.32 → 1.8/2.3 (RCT-anchored tight band).
  • Ladder-to-72: 66–72 = real sizes; เกินมาตรฐาน 🔥 only past 72.
  • B toggles: dashed-line toggles double as the legend, in the bottom-right corner; color-keys move left.
  • Glow: stronger red drop-shadow (≈5+11px), still breathe + reduced-motion guard.
  • Auto-scale: flex-wrap legend row + 44px tap targets, desktop/iPad/mobile.

Acceptance Criteria

  6. Tests green (recalibrated k + ladder-72 + toggle source-grep) + build clean + L1 real-browser.
