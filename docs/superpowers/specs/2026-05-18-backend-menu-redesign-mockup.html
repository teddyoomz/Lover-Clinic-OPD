▼ Themed Bloom Spec
🔥 Dark bloom
Red-Black Space · นทพื้นหลัง linear-gradient(#1a0608, #2c0a0e 50%, #0a0203) + radial red nebula glow + 15+ stars (white·red·orange · sizes tiny/sm/md/lg + glow shadow) + 3 floating embers (animated float-up 4s) + 5 twinkle stars (animated opacity 0.3→1 2.5s). Backdrop blur 8px.
🌸 Sakura bloom
Sakura Petal Rain · พื้นหลัง linear-gradient(#fff0f5, #ffe4ec 50%, #ffd1dc) + radial pink mist + 17-22 petals ขนาดต่างกัน (xs/sm/md/lg) · 3 colors (deep #f9a8d4→#ec4899 · mid #fbcfe8→#f9a8d4 · soft #ffffff→#fce7f3) · animated petal-fall 5-9s loop · rotated 0-340° + translateX drift. Backdrop blur 8px.
🎨 Theme switch
Bloom background reads [data-theme] attribute (Dark=red-space · Sakura=petals) · same component · CSS animation auto-pauses เมื่อ user prefers-reduced-motion · prefers-reduced-motion: reduce → static (no fall/twinkle/float).
📐 Performance
Stars/petals = CSS only (no canvas, no JS loop) · ~20 absolute-positioned divs · CSS animations GPU-accelerated · should add will-change: transform on animated petals/embers · cleanup on bloom-close (unmount animation).
♿ A11y
Bloom overlay role="dialog" + aria-modal="true" · animation honors prefers-reduced-motion · 8 orbs role="menuitem" + tabindex · screen-reader: "เปิดเมนู เลือก 1 ใน 8 หมวด"
🚫 Cosmetic only
ห้ามแตะ flow/logic/wiring · sub-components (BranchSelector · ThemeToggle · ProfileDropdown · StaffChatBubble) wired verbatim · handlers verbatim · state verbatim · test pyramid 6 ชั้น loop จน 100% Perfect
⚡↔📋 Menu Toggle
Desktop + Tablet เท่านั้น (≥768px · mobile <768px จะ force New mode ตลอด) · pill toggle [⚡ ใหม่ | 📋 เดิม] ใน top bar · ตำแหน่งระหว่าง 🛒 shortcut กับ Branch dropdown
