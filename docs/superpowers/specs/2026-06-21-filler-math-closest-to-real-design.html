Purpose / Problem

The penile-filler size simulator was deliberately biased low ("ปัดเศษออกเอาต่ำไว้ก่อน" — floor the condom snap, anchor k at the low end). The user reversed that directive: research as thoroughly as possible, then compute as close to reality as the evidence allows. We ran an exhaustive cited research program — 5-facet baseline → 3-skeptic adversarial verification → a loop-until-dry research loop (9 rounds, converged when a round changed no constant), ~17 agents, every number traced to a primary source (RCTs, large cohorts, meta-analyses, ISO 4074, manufacturer data). This spec encodes the resulting closest-to-real calibration.

Three UX framings were locked with the user before research: (1) the headline shows the PEAK (~1-month) result, with the durable value shown alongside for honesty; (2) the user enters ERECT girth + length (condoms are sized to erect girth); (3) condom snap = FLOOR/round-down (research confirmed this is the correct retention rule) with the next-up neighbor shown.

Decisions (locked via Q&A + research)

  • Framing Q1 = peak headline — user chose "show peak". The girth headline shows the peak (สูงสุด) value only; the durable (ระยะยาว) result is no longer a separate girth line (user revision 2026-06-21) — it surfaces via the condom ระยะยาว–สูงสุด range.
  • Framing Q2 = ERECT input — user enters erect girth+length; the geometric ΔC is computed on the erect base → the (smaller) erect gain. Optional flaccid path converts ×1.25 first.
  • Framing Q3 = FLOOR snap — research confirmed round-down is the correct retention rule (MyONE/ISO); keep FLOOR.
  • Condom = ระยะยาว–สูงสุด range (user revision 2026-06-21) — FLOOR-snap BOTH the durable girth and the peak girth and display the two rungs as a range ("55 – 57 mm", note "ระยะยาว 55 → สูงสุด 57"). Replaces the earlier durable-rung + neighbor display. Thai labels: Durable→"ระยะยาว", Peak→"สูงสุด".
  • Result = condom NUMBERS only (user revision 2026-06-21) — the result card shows just the width numbers ("55 – 57 mm"), NO category name (Super Snug / Regular / Large / Super Wide). The named sizes remain ONLY in the customer's size-pick input dropdown (unchanged).
  • Copy trims (user revision 2026-06-21) — REMOVE the volume-migration warning, the "beyond-mainstream / แทบไม่มีขาย" condom note, the glans decoupled/durability note, and ALL "ช่วงแรก" wording. The math/constants are unchanged; this only changes what the card displays (more marketing-forward, fewer disclaimers — clinic owner's call).
  • Girth k: peak 1.90, durable(12mo) 1.22, full ladder (Yang2020 RCT / Ahn RCT / Wang2022 cohort).
  • Condom ladder corrected to the real MyONE 9-rung [45,47,49,51,53,55,57,60,64]; ceiling 64 (drop fictional 72).
  • Glans ΔØ/cc 0.17 (band 0.13–0.24); discrete 2–3mL saturating; own 85%@5yr fade; decoupled.
  • Volume default 16, range 5–30 (hard-cap 30, warn >20); dose-response saturating via geometry (n=103 regression: more mL ≠ more cm).
  • Defaults re-seeded SE-Asian (erect girth ~10.4cm, length ~13.4cm) — Western seeding overstated girth by ~2 condom sizes.
  • Length: filler increases FLACCID length (+1.65cm durable splint) but not erect/true (+0.6cm fading) — display-only secondary; L in the geometry stays the erect/stretched length.

Acceptance Criteria

  1. fillerMath.estimate with the SE-Asian erect defaults (C0=10.4, L=13.4, V=16, glans=2) returns peak girth gain ≈ +1.3cm and durable ≈ +0.85cm (matches Wang erect anchor), condom as a Durable–Peak range (durable rung 55 – peak rung 57), glans ΔØ ≈ +0.35cm.
  2. Doubling volume does NOT double the girth gain (saturation): gain(32mL) − gain(16mL) < gain(16mL) − gain(0).
  3. Condom uses the real ladder [45,47,49,51,53,55,57,60,64]; the durable + peak girths each FLOOR-snap and display as a range; ceiling 64 (the old +2→72 extension is gone); no "beyond-mainstream" warning copy is rendered. The RESULT shows the width NUMBERS only (no Super Snug / Regular / Large / Super Wide category name); the customer's size-pick dropdown keeps its named sizes unchanged.
  4. Glans dose saturates: ΔØ(3mL) ≈ ΔØ(2mL); glans never changes the shaft girth or condom result; the glans row shows only the ΔØ value (no decoupling/durability note).
  5. Volume slider is hard-capped at 30mL (the old 50cc ceiling is gone); NO warning banner is rendered at any volume.
  6. The geometry L is the erect/stretched length; the flaccid-length splint gain is display-only and never enters the girth calc.
  7. Girth headline shows the peak (สูงสุด) value only — no separate durable girth line; the durable result surfaces only via the condom ระยะยาว–สูงสุด range. The words "ช่วงแรก", "Durable", "Peak", "splint" appear nowhere in the user-facing strings (Thai labels ระยะยาว / สูงสุด / พยุงไม่ให้หด are used instead).
  8. Risks / Open Questions

  • Erect-efficiency is a model extrapolation — the k-ladder was fitted on flaccid-shaft geometry; Wang's erect serial data (n=38) is the main erect anchor. Documented as a caveat, not hidden.
  • Thai girth data gap — defaults use the Vietnamese n=14,597 regional proxy; no rigorous Thai girth cohort exists. Acceptable for a toy; disclosed.
  • Individual variance dwarfs the point estimate (SD ±0.95–2.2cm) — mitigated by the ±1cm band + "estimate not promise" copy.
  • HA-specific — calibration is for hyaluronic acid; PLA/PMA differ. The sim assumes HA (the clinic's product).
  • Honesty vs marketing (owner decision 2026-06-21) — peak is the headline but the durable value is still shown (the "คงเหลือ ~12 เดือน" line + the condom Durable–Peak range), so the lasting result is never hidden. However, at the owner's request the volume-migration warning, the beyond-mainstream condom note, the glans decoupled/durability note, and the "ช่วงแรก/transient" framing are removed — the toy is more marketing-forward with fewer medical disclaimers. The underlying math stays evidence-accurate; only the displayed copy is lighter.
  • Test churn: ~98–102 existing filler tests lock the old constants → expect a batch of V21-style updates alongside the new tests.
