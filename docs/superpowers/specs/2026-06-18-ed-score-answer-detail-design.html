Purpose / Problem

ในกล่อง "สมรรถภาพ · ED Score" (CustomerDetailView) แต่ละช่อง (ADAM / IIEF-5 / MRS / PE) ตอนนี้แสดงแค่คะแนนรวม. แพทย์อยากกดเข้าไปดูว่า ลูกค้าตอบแต่ละคำถามว่าอะไร เพื่อรายละเอียดเชิงคลินิกที่มากขึ้น "เหมือนกับ box บันทึกข้อมูลรับเข้าใน frontend ที่แสดงรายละเอียดครบ" (= modal ดูข้อมูลคิวใน AdminDashboard ที่ render renderAdamSection/renderMrsSection/renderIiefSection). ทำให้แต่ละช่องกดได้ → เปิด modal คำถาม+คำตอบรายข้อ.

Acceptance Criteria

  1. กดช่อง IIEF-5 ที่มีค่า → เปิด modal แสดง 5 คำถามเต็ม + คำตอบเป็น option label (เช่น iief_1=3 → "ปานกลาง (3)") + score banner 13/25 + interpretation.
  2. กดช่อง ADAM → 10 คำถาม, ข้อที่ true → "มีอาการ" (ส้ม), ข้อ false/ไม่มี → "ไม่มี".
  3. กดช่อง MRS → 11 คำถาม, ค่า → "ระดับ N — <label>" (0 = "ระดับ 0 — ไม่มีอาการ").
  4. กดช่อง PE (เมื่อมีอาการ) → modal 1 ข้อ "มีอาการ".
  5. ช่องที่เป็น "—" (ไม่มีข้อมูล) กดไม่ได้ (ไม่เปิด modal).
  6. เลือกครั้งเก่า (round-select) แล้วกดช่อง → modal โชว์คำตอบของครั้งที่เลือก (ไม่ใช่ครั้งล่าสุด).
  7. ปิด modal ด้วย ✕ และ ESC ได้; กด backdrop รอบๆ → modal ไม่ปิด (AV78).
  8. ช่องเป็นปุ่มจริง (focusable, Enter/Space เปิด modal ได้); ดาร์ก+ไลท์ธีม modal อ่านออกทั้งคู่.
  9. buildEdAnswerRows map ค่า adam/iief/mrs/pe ถูกต้อง + edge (undefined → "—", mrs 0 → "ไม่มีอาการ", iief out-of-range → "—").
