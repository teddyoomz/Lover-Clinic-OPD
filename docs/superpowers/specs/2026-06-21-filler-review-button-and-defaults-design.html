Date: 2026-06-21 · Status: Approved by user (Q&A locked) · Scope: the public ?play=filler page only · Plan: docs/superpowers/plans/2026-06-21-filler-review-button-and-defaults.html

Purpose / Problem

Three owner-driven changes to the public filler simulator. The simulator is a lead-gen toy — the owner wants a prominent, on-theme CTA to funnel players to the clinic's real before–after review pages, plus two small calibration tweaks.

  1. Review CTA (the design item). A single eye-catching button that, to save space, opens a popup to choose which review set to view — circumcised (https://loverclinic.com/review-enhance-circum/) vs uncircumcised (https://loverclinic.com/review-enhance-nocircum/). It must be beautiful/on-theme, prominent, and enticing.
  2. New defaults: length 5 inches (12.7 cm), total filler 10 cc.
  3. Thinner red dashed "after" outline: reduce its stroke width by a further 50% (1.7 → 0.85).

Decisions (locked via Q&A with user)

  • Q1 = A: review button placed UNDER the results card, above the contact-buttons row (peak-interest moment).
  • Q2 = style 1: fire gradient red→gold + glow + gold border (the most prominent / on-theme).
  • One button → popup with 2 choices (circum / nocircum) to save space (owner). Each opens the matching loverclinic.com review URL in a new tab.
  • Defaults: length 5 in (12.7 cm), filler 10 cc.
  • Red dashed outline: −50% thickness (1.7 → 0.85).

Acceptance Criteria

  1. A prominent fire-gradient "ดูรีวิวจริง" button renders directly below the results card + above โทร/LINE/FB.
  2. Click → a dark fire-bordered popup with two choices; "แบบขลิบ" opens …/review-enhance-circum/ and "แบบไม่ขลิบ" opens …/review-enhance-nocircum/ in a NEW tab (noopener,noreferrer); X/backdrop/ESC close it.
  3. On first load the length slider reads 5.0 นิ้ว and the filler slider reads 10 cc.
  4. The red dashed "after" outline strokeWidth is 0.85 (no 1.7 remains).
  5. Theme/layout/math otherwise unchanged; full vitest green; build + verify:filler + obfuscation clean; Rule Q-vis L1 matches this mockup.

Risks / Open Questions

  • External URLs are hardcoded to loverclinic.com — if the review paths change, update the constant. Open in a new tab so the toy session is preserved.
  • Default 10 cc / 12.7 cm changes the default displayed result (smaller than the prior 16 cc / 13.4 cm) — intended (owner's chosen defaults).
  • Modal backdrop-close is allowed here (public toy, no form) — differs from the OPD AV78 explicit-close rule, which applies to data-entry modals, not this 2-choice picker.
