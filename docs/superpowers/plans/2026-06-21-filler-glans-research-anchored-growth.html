Derived constants (no guessing — locked here)

Research anchors @ 2 cc (Kwak/Kim, PMC4550597 + PMC7807328):
  peak (6mo)    +16.58 mm circ → ΔØ +0.528 cm → on ref glans Ø 3.5cm = +15.1%
  durable (5yr) +14.10 mm circ → ΔØ +0.449 cm → on ref glans Ø 3.5cm = +12.8%  (16.58×0.85=14.1 ✓)

Cube-root: ΔØ% = (1 + 2/veff)^(1/3) − 1  →  solve veff at 2 cc:
  peak    : 1.151^3 = 1.5247 = 1+2/veff → veff_peak    = 3.81   (381/100)
  durable : 1.128^3 = 1.4354 = 1+2/veff → veff_durable = 4.59   (459/100)

Sanity (dg0=3.5): 2cc → durable +0.45 / peak +0.53cm ✓ ; 10cc → +1.65 / +1.88cm ; 15cc → +2.18 / +2.46cm (monotonic, no plateau).

// Filler IS the added volume (peer-reviewed mechanism); the head is compact → Ø grows as the cube
// root of total volume → never plateaus, each cc adds a little less Ø. Anchored to the only published
// glans dose (Kwak/Kim 2cc): durable +0.45cm Ø (5yr, +14.1mm circ) / peak +0.53cm Ø (6mo, +16.6mm circ).
// veff = effective glans filler-compliance volume, calibrated so 2cc lands on those anchors at a typical

// see GLANS_CAVEAT in fillerRefs.js. Integer fractions so the build obfuscator hides the literals.
export const GLANS_FILL_VOLUME_CC = { peak: 381 / 100, durable: 459 / 100 }; // cc
export const GLANS_SPLIT_MAX_CC = 15; // head-injection cap (slider) — locks the exaggeration
export const GLANS_CC = { min: 0, max: 15, step: 0.5, default: 0 };
// initial glans baseline = ratio of shaft Ø; default 1.0 = shaft Ø.
export const GLANS_BASE_RATIO = { min: 0.75, max: 1.25, step: 0.05, default: 1.0 };
export const glansDiameterGain = (glansCc, dg0, veff) => {
  const d = Math.max(Number(dg0) || 0, 0);
  const cc = Math.max(Number(glansCc) || 0, 0);
  const v = Number(veff) || 1;
  return d * (Math.cbrt(1 + cc / v) - 1);
};
