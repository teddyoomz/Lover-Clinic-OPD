Flow

cron 00:30 UTC (CRON_SECRET)→ reads: scheduled_task_status · recon-daily-{y} · push_config/tokens+settings · error count 24h · system_config→ evaluateInfraHealth() pure→ write infra-health-latest + history + task status→ warn/red → staff card + LINE push→ purge client_error_log >30d
Beacon — capture → sanitize → gate → POST → validate+cap → store → viewer/health-count
onerror / unhandledrejection / ErrorBoundary→ sanitize (PHI-safe URL) + dedupe 1/5min/hash + max 20/session→ POST /api/client-error (sendBeacon→fetch keepalive)→ validate + truncate + tx daily-cap 500→ client_error_log (default-deny)→ admin list endpoint → viewer · health cron count()
