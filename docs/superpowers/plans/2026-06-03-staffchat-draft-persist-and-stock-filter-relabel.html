Staff-Chat Draft Persist on Minimize + Stock Filter Relabel — Implementation Plan

For agentic workers: Execute inline (executing-plans). Tightly-coupled small UI work; subagents thrash on this codebase baseline (V81/V86/V159 lesson). Steps use checkboxes for tracking.

Goal: (A) Preserve in-progress staff-chat work (text + reply + staged image/file uploads) when the chat is minimized with "–", clearing only on Frontend↔Backend switch / reload / tab-close. (A-bis) Show a distinct draft indicator on the minimized bubble. (B) Relabel + reorder the stock balance หมด/ติดลบ filters.

Architecture: Hide-don't-unmount — the chat Panel stays mounted (display:none) on minimize so the Composer's local state never dies. Composer reports a single hasDraft boolean up via onDraftChange (heavy draft stays in the composer); the Widget passes it to the Bubble, which shows a dark-zinc ✏️ badge top-left (distinct from the white/red unread count top-right). Feature B = pure JSX relabel + block swap. All changes additive.

Tech Stack: React 19, Tailwind, lucide-react, Vitest + RTL. Spec: docs/superpowers/specs/2026-06-03-staffchat-draft-persist-and-stock-filter-relabel-design.html.

export function StaffChatComposer({ onSend, recentMentionCandidates = [], replyingTo, onClearReply, onPrepareAndUpload, onSendSticker, onDraftChange }) {
  // ...existing state...

  // Report draft presence up so the minimized bubble can show an indicator.
  // hasDraft = unsent text OR ≥1 staged file OR an active reply. setHasDraft
  // from the parent is stable; same-value calls no-op (React bails). The heavy
  // draft (text/pendingFiles/File-objects) stays here — only a boolean lifts.
  useEffect(() => {

export function StaffChatBubble({ unreadCount, hasDraft, onClick }) {

    <button
      type="button"
      onClick={onClick}
      data-testid="staff-chat-bubble"
      className="fixed bottom-[88px] right-3 md:bottom-4 md:right-4 w-14 h-14 rounded-full bg-rose-600 hover:bg-rose-500 text-white shadow-xl hover:shadow-2xl transition-all hover:scale-105 flex items-center justify-center z-[9000]"

    >

          the unread badge (white/red, top-right) so the two never confuse. */}
      {hasDraft && (
        <span
          data-testid="staff-chat-bubble-draft"
          className="absolute -top-1.5 -left-1.5 w-[21px] h-[21px] rounded-full bg-zinc-900 border-2 border-zinc-600 text-[11px] flex items-center justify-center"
