ออกแบบเฉพาะแถบเมนูบน เลือกแบบที่ชอบแล้วผมจะ implement.

Variant A

Compact Pill Bar — Linear/Vercel style

แถวเดียวบน desktop · มี 5 tabs หลักโชว์ + overflow ⋯ สำหรับ ตั้งค่า/หลังบ้าน · จองรวมเป็น dropdown · utilities ย่อเป็น user-menu (👤▾) ภายในมี theme + online + ออก. มือถือ = bottom nav 5 ช่อง + top bar กระชับ.

Variant B

เคล็ดลับ: active underline = editorial vibe พรีเมียม
Variant C

เคล็ดลับ: content area สูงที่สุดในทุก variant
Variant D

Top bar แสดงเฉพาะ "Brand ▸ Active tab" + ⌘K search bar + actions ขวา · ไม่มี tabs แสดงในแถบ — ทุกอย่างเปิดผ่าน command palette (กด ⌘K / Ctrl+K หรือคลิก search bar). มือถือ = floating 🔍 ปุ่มล่างขวา.
