1 รอบวงลำตัว (girth)

ขั้น A · เรขาคณิต — ทรงกระบอกรอบวง C มีพื้นที่หน้าตัด A = C²/4π. ฉีดปริมาตร V กระจายตามยาว L ทำให้พื้นที่เพิ่ม ΔA = V/L → รอบวงใหม่:

g = √(C₀² + 4π·V/L) − C₀

ขั้น B · คูณตัวปรับ k (เนื้อเยื่อจริงขยายมากกว่าเรขาคณิตล้วน) → ได้เป็น ช่วง คงตัว→พีค:

Δรอบวง = k · g   ( k = 1.22 คงตัว · 1.90 พีค )

💡 insight: เพราะมี V/L → ลำตัว สั้น (L น้อย) ฉีด cc เท่ากัน รอบวงขึ้น เยอะกว่า · k คาลิเบรตจาก RCT [1][3]

2 เส้นผ่านศูนย์กลาง + ขนาดถุงยาง

Ø = รอบวง / π   ·   ความกว้างถุงยาง (mm) = รอบวง(cm) × 5

มาตรฐาน ISO 4074 [5]: nominal width = ครึ่งรอบวง · ผลลัพธ์โชว์ mm ดิบ (ปัดเศษ) แล้ว snap แบบ floor = ไซส์ใหญ่สุดที่ยังกระชับ (กันหลุด)

3 ส่วนหัว (glans) — โมเดลคงปริมาตร (cube-root)

หลักการ ฟิลเลอร์ = ปริมาตรที่เพิ่มเข้าไป (งานวิจัยยืนยัน) · หัวเป็นทรงตัน → Ø โตตามรากที่สามของปริมาตร = โตเรื่อยๆ ไม่ตัน แต่ละ cc เพิ่มน้อยลง · ไม่กระทบรอบวง/ถุงยาง · ล็อคสูงสุด 15cc

Ø หัว = Ø₀ × ∛(1 + cc/veff)
veff = 4.59 (คงตัว) · 3.81 (พีค) · คาลิเบรต 2cc → +0.45/+0.53cm Ø

anchor: 2cc → หัวโต +14.1–16.6 มม.รอบวง (Moon 2015 [6]) · เกิน 3 mL = ประมาณตามหลักคงปริมาตร (ยังไม่มีวิจัยหัวเกิน 3mL) สอดคล้องกับวิจัยลำตัว ~20mL ที่โตต่อเนื่อง

4 ความยาวตอนอ่อน (flaccid) — ผลพลอยได้

เพิ่ม = MAX·(1 − e^(−cc ลำตัว/10))  MAX = 2.0 / 3.0 cm

เป็น splint กันหดเข้า (anti-retraction) ไม่ใช่การยืดจริงตอนแข็งตัว · เฉพาะ cc ลำตัว (ส่วนหัวไม่ช่วย splint) [2]
