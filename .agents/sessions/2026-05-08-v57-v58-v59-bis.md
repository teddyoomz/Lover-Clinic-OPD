## Lessons (full reasoning → v-log-archive.md when V57/V58/V59 V-entries are filed)

- **TDZ in useMemo deps**: useMemo deps array evaluates synchronously at hook setup. References to variables declared later in the component throw silently in React. Black screen + only generic component error in console. New hooks should default to landing AFTER all state/memo declarations they reference.

- **Schema-vs-consumer drift class**: a downstream filter implies a schema field; if the validation file never declares it, legacy data fails the filter silently. AV30 invariant: defensive `?? default` in consumers + schema declaration day-one.
