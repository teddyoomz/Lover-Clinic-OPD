### Main form fields
| Field | Type | Required | Notes |

### Sub-item pattern (repeated for all 4 categories)
Common per-item fields: `qty` (number, min 1 or 0.01), `price` (number, min 0), `item_discount` + `item_discount_type` (radio %/บาท), `is_vat_included` (checkbox).

Category-specific:
- **Courses** (f2): + `course_name` (text, searchable)
- **Products** (f3): + `product_name` (text) + `is_premium` (checkbox = ของแถม)
- **Promotions** (f4): + `promotion_name` (text)
- **Takeaway meds** (f5): + `takeaway_product_id` (select2) + `is_premium` + medication detail fields:
  - `generic_name` (text)
  - `indications` (text)
  - `dosage_amount` (text) + `dosage_unit` (select: เม็ด/ซีซี/ช้อนชา)
  - `times_per_day` (text)
  - `administration_method` (radio: ก่อนอาหาร 30 นาที / หลังอาหาร / ทุกๆ N ชม.)
  - `administration_method_hour` (text, required if method = ทุกๆ)
