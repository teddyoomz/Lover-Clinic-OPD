# Design Context — LoverClinic Backend Dashboard

## Target Audience
- ทุกคนในคลินิก: เจ้าของ, แพทย์, พยาบาล, รีเซฟชัน, ผู้จัดการ
- ใช้งานรายวัน: จองนัด, บันทึก OPD, ขายคอร์ส/ยา, ดูข้อมูลลูกค้า
- ต้องทำงานเร็ว กดน้อย อ่านง่าย

## Brand Personality & Tone
- **Dark + Powerful + Fire** — แดง ดำ ขาว ไฟ เข้ม เท่ มีพลัง
- Premium masculine aesthetic (คลินิกผู้ชาย)
- สีแดงห้ามใช้กับชื่อ/HN ผู้ป่วย (วัฒนธรรมไทย = สีคนตาย)
- สีทองห้ามใช้ (user ไม่ชอบ)
- Accent color: customizable per clinic (default #dc2626 red)

## Design System
- **Theme**: Dark (primary) + Light mode supported via CSS variables
- **Dark palette**: bg-base #050505, bg-surface #0a0a0a, bg-card #0f0f0f
- **Accent colors per tab**: violet (clone), teal (customers), amber (master data), sky (appointments), rose (sales)
- **Typography**: System sans-serif, Thai language primary
- **Framework**: React 19 + Tailwind CSS 3.4
- **Icons**: Lucide React

## Key Use Cases
1. Clone ข้อมูลลูกค้าจาก ProClinic
2. ดูข้อมูลลูกค้า (3-column detail: profile | medical | courses)
3. จัดการข้อมูลพื้นฐาน (sync products/doctors/staff/courses)
4. นัดหมาย (resource time grid: rooms × time slots)
5. ขาย/ใบเสร็จ (CRUD + payment management)
6. สร้างการรักษา OPD (full medical form + billing)

## Priority
- ใช้งานง่าย + อ่านง่าย + หาเจอเร็ว + กดน้อย
- ดูดี มีเอกลักษณ์ ไม่เหมือน template ทั่วไป
- Balance ทั้งสองอย่าง
